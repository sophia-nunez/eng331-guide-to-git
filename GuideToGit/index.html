What Is Our Guide to Git

Learn how to use git, whether you're a student working solo or need it for group projects.

Git is used for source control in projects around the world. This guide focuses on the basics of git and how to use it when working with others. This website contains information and examples, and check out the GitHub repository for some practice exercises!

  • Basics
  • Learn the basics of git. This section is for complete beginners - learn how to clone a repo, push, and pull your code.
  • Group
  • Know the basics but working in a group for the first time. Learn about the workflow and how to share your code when working with others in a repository.
  • Extra
  • Beyond the basics of group work. Want to learn how to manage branches, or have a new merge conflict to handle? Additional topics or issues you may run into can be found here.
